# Cow say poor edition

## Getting started

**First of all**

```git clone https://github.com/MatheusVict/CowSay-poor-edition.git```

**After that**

```cd CowSay-poor-edition```

**And Finally**


```shell
./cowsay.sh "your message here bro"
```

**Output**

